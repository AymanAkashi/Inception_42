Docker

is an open platform for developing, shipping, and running applications

### what is Docker Docker is a platform for building, running and shipping applications in a consistent manner so if your application works on your development machine it can run and function the same way on other machines. let's go deeper, we have two isolated environment allows multiple application and different versions of some software, so one application use node 14 another application may use node version 9. both applications can run side by side on the same machine without messing with each other, this is how docker allows us to consistently run application on different machines.
